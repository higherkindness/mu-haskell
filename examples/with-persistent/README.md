# with-persistent

## Execution

Running the server:

```bash
$ stack run persistent-server
```

In another terminal, run the client:

```bash
$ stack run persistent-client add "Flavio" 28
```

Alternatively, you can also use the record version:

```bash
$ stack run persistent-client-record watch
```
